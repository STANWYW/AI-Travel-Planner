## 🚀 如何启动

### 最简单的方式（推荐）

```bash

cd /home/stan/code/AITravelPlanner

# 2. 运行启动脚本
bash START_APP.sh
```

这会自动：
- 启动 Docker 后端和数据库
- 安装前端依赖（如需要）
- 启动前端开发服务器

然后访问：**http://localhost:5173**

### 手动启动

```bash

# 启动前端（新终端）
cd frontend
npm install
npm run dev
```

## 🔑 API Key 配置

### 必需配置

**OpenRouter API Key**（用于 AI 功能）

1. 访问：https://openrouter.ai/
2. 注册并充值 $5-10
3. 创建 API Key
4. 在应用"设置"页面输入

**或使用阿里云百炼平台的 API Key**（助教批改使用）

### 可选配置

- **科大讯飞语音识别**: 配置后启用高级语音功能
- **高德/百度地图**: 配置后启用地图展示

**说明**: 没有这些可选 API 也能正常使用基础功能
